# flutter-note-app

User-friendly note-taking application built with Flutter, enabling seamless note sync, signup and login using Firebase and Firebase Authentication

## Prerequisites

- Flutter SDK
- Android Studio
- Android SDK

## Running the Project

- Clone this project
  `git clone < project-url.git >`
  
- Install dependencies
  `flutter pub get`
  
- Run
  `flutter run`
